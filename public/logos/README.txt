LOGO IMAGES INSTRUCTIONS
========================

Please save your logo images in this directory with the following names:

1. logo-main.png
   - This is the main ЦТТ logo (the geometric blue logo)
   - Recommended size: 64x64px to 128x128px
   - Format: PNG with transparent background

2. logo-text.png
   - This is the text logo "Центр трансфера технологий при КГТУ им. И. Раззакова"
   - Recommended size: Width 300-600px, height proportional
   - Format: PNG with transparent background

The header component will automatically use these images once they are placed here.
If the images are not found, it will fall back to text-based logos.
